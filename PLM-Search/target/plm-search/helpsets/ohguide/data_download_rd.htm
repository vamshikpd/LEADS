Data Download

The Data Download link is located just below the Step 1 – Case#/Reason action bar. This download function is intended to provide a data access method for LEADS 2.0 users with specific privileges to download complete parolee information for one or more parolees. This function requires the user to be included in the “Data Download” group. This privilege must be assigned by either the LEA admin or a CDCR admin. This function groups the results into files based on the particular attributes of the select parolees. For instance all parolee address are group into an “address” file. Once these files are created they are packaged into a zip file and staged for later download. The data download process is a three part step. The first step prompts the user to enter a “Last Download Date”. Once the date is entered and validated a request is submitted to construct the results. Once this is complete a link to the download zip file can be accessed by selecting the data download link.

To download data:

Click the data download link and a form is displayed with the message below, the Last Update Date data field, the ability to view Help documentation, and the Download, Clear Fields and Cancel action buttons.

The Help icon is displayed in the top right hand corner of the Data Download dialog drop-down box.

Click the help icon so you can view Help documentation related to the Data Download function or the entire LEADS2 application.

Enter the Last Update Date (mm/dd/yyyy).

 

The Last Update Date relates to when the data was last updated in LEADS. The date cannot be greater than 100 days.

 

Example

If today’s Date is April 1, 2010. You can use any date after December 21, 2009 and prior to April 1, 2010.

 

Click the Download action button or press the <Enter> key to activate your data download and the following pop-up warning message appears: “Data download has been submitted and is in progress. Please select the ‘Data Download’ link the downloaded data and photos”.

Select the Data Download link when you want to check if your download has completed. Click the link at the top of the form that starts with “data” line to continue your download. The download dialog box appears with the following:

•            A question: “Do you want to open or save this file?”

•            A file with a default name created by LEADS2

•            Where the file was created

•            Three action buttons: Open, Save and Cancel

•            A risk information message: “While files from the Internet can be useful, some files can potentially harm your computer. If you do not trust the source, do not open ore save this file?

•            A “What’s the risk?” information link that explains the possible risks to your computer.

If you first want to view your data, click the Open action button. Then save your file to a folder where you can easily retrieve the data at a later time or day.

To save your download, click the Save action button and the Select a Destination dialog box appears with:

•            A note requesting you to “Select a folder to extract files to:”

•            A note letting you know the “Files will be extracted to this directory”

•            A Browse and a Password action button

•            The word ‘Extracting’ and a field  that will show you the file(s) being extracted

•            A Next> and a Cancel action button

 

Enter the name of the folder/directory so you can easily retrieve the data at a later time or date. If you’re not sure of the folder name, click the Browse action button to find and select your folder.

Click the Password action button if you want to password protect your downloaded data. Be sure to use a password that is easy to remember and don’t forget to store it in a safe place (just in case).

Click the Next> action button to activate the file extraction process. The Extraction Complete dialog pop-up box appears with following information:

•            The message “Files have been successfully extracted to the following directory

•            The name of the folder/directory you selected

•            The message “To see your extracted files, check the box below:

•            The check box with the note “Show extracted files”

•            The note that asks you to “Press finish to continue.”

•            A Finish and Cancel action button.

Click the Finish action button and you downloaded files are saved.

If you viewed the data in the file and you decide you don’t need to download the information, click the Cancel action button.

If you don’t enter a Last Update Date or you don’t use the correct format, the following error message displays: Error: “Please enter date in mm/dd/yyyy format”.

If you enter a Last Update Date and no data is available the following message will appear.

 

The message below the two red error lines is the same as the message that was displayed when you clicked the Data Down option.

If you attempt to enter a Last Update Date for the previous 100 days the following error message displays: Error: “Photo (weekday mmm dd hr:mm:ss yyyy: Last Update Date can not be greater than 100 days)”.

If you attempt to activate a second Data Download the following error message displays: Error: “Data download is already in progress. Please wait until the current process completes before requesting a new download”.

If you previously downloaded data and information data download drop-down box will display your recent download(s), download exceptions or download problems you may have had when performing data downloads.
