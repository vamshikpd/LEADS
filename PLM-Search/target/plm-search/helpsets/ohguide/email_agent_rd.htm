Email Agent

The E-mail Agent function is available for a LEADS 2.0 user to communicate directly to the Parole Agent of record for any given parolee. This function is accessed by clicking on the Agent e-mail address from the parolee detail page. If the agent e-mail address is not available the link will not be displayed.

To send a communication directly to the parole agent of record, select a parolee from the parolee results page. Once the parolee detail page is displayed click the “Agent e-mail” address. This will display the “EMAILAGENT” page. The e-mail address of the current LEADS 2.0 user will be pre-populated and cannot be changed. The e-mail address of the parole agent will also be pre-populated and cannot be changed. Additionally the subject line of the e-mail will filled in with the CDC Number and Name of the selected parolee.

After entering message text click the “Send Email” button. To cancel click the “Cancel” button. The email will be forwarded directly to the parole agent.
