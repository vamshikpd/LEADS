Frequently Asked Questions (FAQ)

Version LMS 6.0 UZH

6. Configuration of a Course

How do I configure course elements?

Please have a look at the user manual, chapter 9. All course elements will be explained along with their necessary configuration.

What is the difference between visibility and access?

The access to course elements can be restricted in two ways. By restricting the visibility of a course element that element will no longer be visible in the course navigation. By restricting the access that element will still be visible in the course navigation but when clicking on it a message will appear stating that this element is not accessible.

Where can I modify that message that a course element is not accessible?

In the tab “Visibility” you can adapt that message at the bottom of the page.

What does the option “Apply rules also for owners and tutors” mean?

When activating this option visibility and access rules also apply for course authors and tutors of learning groups. If this option is not activated those rules do neither effect course authors nor tutors of learning groups.

Is it possible to let course participants register in two learning groups at the same time?

It is only possible to register in one group per registration element. If course participants should be able to register in several groups you have to provide them with several registration elements. You can also add members to a certain group manually by means of the group management.

What does the message “No … selected yet” in the course editor mean?

There are certain course elements in order to embed a learning resource into a course. For example, a Wiki has to be created first in learning resources (select “Learning resources” in the main navigation, then click on “Wiki” in the tool box “Create” on your right) before embedding it into the LMS course along with the corresponding course element. This enables you to use the same learning resource in several LMS courses.

I want to use some files in several courses at the same time. How do I proceed?

Please create a resource folder in learning resources before uploading your files. In the detailed view of your course you then select this resource folder in the tab “Resource folder,” option “Modify settings.” Your files from that folder are now stored in the subfolder “Shared folder” of the course editor. You can embed these files into your course.

How do I upload my files into the resource folder?

Click on the title of the resource folder, section “My entries.” In the tool box “Learning resources” on the top right you will see the link “Edit content.” By means of the link “Add” you can now upload your files into the resource folder.
