Frequently Asked Questions (FAQ)

Version LMS 6.0 UZH

4. Information for Students

4.1 How to Take a Course

How do I find an LMS course?

In LMS’s main navigation you have to select “Learning resources” before looking for your course in the catalog, either via “Search form” or via “Courses” in the list containing all available LMS courses.

Why bookmarks and where can I set them?

Bookmarks are shortcuts to e.g. an LMS course. You do not have to look for courses over and over again in the catalog or by means of the search form. In your Home, section “Bookmarks” you will get a firsthand link to your course. On the course’s screen you will find a link to bookmark a course by using the tool box on the right, option “General.”

Why can’t I find my course in learning resources?

Perhaps this course has not been published yet. Please contact the course author.

Why can’t I find a certain course module or why don’t I get access?

Perhaps the access to certain modules is restricted. Please turn to the course author.

I wanted to start a course but all I got was the error message "You do not have enough access rights to start this learning resource." What can I do?

That course does exist in LMS. However, it is not accessible for all LMS users. Please contact the course author.

4.2 Enrollment

How can I enroll in a certain learning group?

Please select the link “Registration” right next to the group you want to join. In case there are no more vacancies there still might be a waiting list. Only one group per enrollment is allowed.

What can I do if there are no vacancies in any learning group?

Please contact the course author to modify the course settings.

I’m still enrolled in out-dated learning groups. How do I cancel my enrollments?

There are two possibilities:

  1. You can do this on your own if the course author has provided you with this option. Open the course, navigate to “Registration” and select the link “Cancel.”
  2. If that course or its registration has been blocked in the meantime you have to ask the course author to assist you in cancelling your enrollment.

4.3 Project Groups

How do I find project groups?

At the moment it is not possible to search for project groups. Project groups are created by users and only visible for group members. You can find your groups by using the main navigation, tab “Groups.” If a project group that you are member of is not visible it might be that you are not registered as a member yet. Please turn to the owner of that project group.

How can I create project groups?

In the tab “Groups” you have to select “Create” in the tool box on your right and then “New project group.” Provide your group with a name, configure some group tools, and invite other LMS users to join your group.

How can I cancel my enrollment in a project group?

Select the link “Quit” in the tab “Groups” next to the corresponding group.

How can I delete a project group?

If you own that group there will be a link to delete your project group in the tab “Groups” right next to the group’s name.
