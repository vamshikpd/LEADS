Save Search

The Save Search function allows you to save searches you know you’ll be using on a regular basis.

Example

You always need to perform a Criminal Analysis for all active parolees in Parole Units under your jurisdiction (Bakersfield 2).

 

 

 

 

The Search page always lists all parolees in the database when you log into the LEADS2 application or you hit the Search button next to the Keyword Search function.

 

 

 

Follow the Keyword Search or Refine by instructions to develop the search criteria you wish to save. When you created your search criteria you can return to this Save Search via the cross-reference links located at the top or end of either section.

You can navigate to or from either section:

To Keyword Search

To Refine by

 

To jump to the Keyword Search or Refine by section, do the following:

1    Press and hold the <Ctrl> key.

2    Move your cursor over the cross-reference link To Keyword Search or To Refine by.

3    Click the link. Your cursor gets placed in front of the cross-reference header you selected.

Example

Press and hold the <Ctrl> key; move your cursor over the cross-reference link Keyword Search and click. The cursor is now located in front of the Keyword Search header.

To return from the Keyword Search section to this Save Search section:

1    press and hold the <Ctrl> key

2    Move your cursor over the cross-reference link To Return to Save Search.

3    Click the link. Your cursor is placed in front of the Save Search header

 

The to-and-from cross-reference jumping process works the same for all cross-reference links are located in the document.

 

 

 

Return cross-reference link(s) are located just above the header line at of a chosen section and below the last sentence at the end of that section.

You are ready to save your search criteria.

•            Click the Save Search icon and the Save Search dialog box appears.

•            Type the name a name for the search criteria you want to save. Make it meaningful so that you can quickly find it at a later time.

•            Click the Save action button to activate the save function.

•            Click the Clear Fields action button if you want to clear your selection. You’re returned to the main Search page

•            Click the Cancel action button and your search criteria won’t be saved. You’re returned to the main Search page

Delete your saved search criteria:

•            Click the Delete Saved Search icon and the Delete Save Search dialog box appears with this question: Are you sure you want to delete?

•            Click the Save action button to save your search criteria.

•            Click the Cancel action button or the close icon (top right corner of the dialog box) and your Saved Search will not be deleted. You are returned to the main Search page.
